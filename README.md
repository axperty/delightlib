### Overview

Delight Lib is a library for Farmer's Delight add-on developers. It makes the creation of food items, knives, cabinets, and recipe management so much easier.

### Contributing

Your contributions to this project allow Delight Lib to be improved to deliver a better gameplay experience for all players. Feel free to submit your code suggestions or bug reports.

### Features

When using Delight Lib, you can easily implement the following blocks and items with simple lines of code:
- Food items and placeable food items
- Cabinets
- Crates and bags
- Knives
- Crops

### Create your own add-on for Farmer's Delight

For a detailed guide on how to easily create your own add-on for Farmer's Delight, [check the documentation here](https://github.com/axperty/delightlib/wiki).
