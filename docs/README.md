# Welcome to the official Delight Lib documentation! 

This mod is a utility library for Farmer's Delight add-on developers that makes it a lot easier to create food items, knives, creates and bags, cabinets, and custom recipes with a few lines of code, letting Delight Lib to do all the hard work.
